CSS

We are going to learn CSS - Cascading Style sheets
  • CSS describes how html elements are displayed on screen
  • To specify CSS. we make use of tag
  • Between the style tags we use CSS that is applied to the webpage

Syntax in CSS

There are 3 parts css is made up of
  1. selector
    • It is used to describe to which we need to apply style
  2. property
    • Attribute that includes fontsize, width, height, background colour etc..
  3. value
    • Value for the property
